2023

Fundamentos em AWS

Formação em AWS pela Escola da Nuvem. A AWS é um serviço de computação em nuvem oferecido pela Amazon, que permite realizar uma ampla gama de atividades, como armazenamento, processamento de dados e implantação de aplicações. Durante o treinamento, aprendi sobre os principais serviços da AWS, incluindo EC2, S3 e RDS, além de adquirir conhecimentos sobre arquitetura de soluções, segurança na nuvem e práticas de DevOps.

2024

Desenvolvimento Full Stack

Na CodePlus, estou aprofundando meus conhecimentos em programação Java, incluindo o uso do Spring Boot e todo o ecossistema do framework Spring. Além disso, estou estudando padrões de arquitetura de software, como os princípios SOLID, entre outros. Atualmente, estou desenvolvendo um sistema de votação, o que me permite aplicar e consolidar minhas habilidades em Java, Spring Boot, HTML, CSS, JavaScript e AWS.

Meus Projetos

Sistema de Chamados

Sistema construído para a disciplina de Programação Orientada a Objetos, permitindo o gerenciamento de usuários e chamados abertos pelos mesmos.

O sistema é voltado para a gestão de pautas de votação em assembleias e decisões importantes, abordando as necessidades e regras de negócio específicas desse contexto. Até o momento, foram implementadas várias funcionalidades e há um planejamento contínuo para o desenvolvimento futuro.

Versão Web do Supp0rt4All

Estou desenvolvendo uma versão web do sistema de chamados, que atualmente está em fase de preparação. Em breve, esta versão estará disponível para uso, oferecendo uma plataforma acessível e intuitiva para a gestão de chamados e suporte técnico. A versão web visa proporcionar uma experiência de usuário aprimorada, com todas as funcionalidades necessárias para o gerenciamento eficiente de chamados.
